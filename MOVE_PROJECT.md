# Инструкция по перемещению проекта

## Проблема

Flutter компилятор шейдеров (impellerc) не может работать с путями, содержащими кириллицу (например, "Документы"). Это известная проблема на Windows.

## Решение: Переместить проект

### Шаг 1: Создайте новую директорию без кириллицы

Создайте новую папку в одном из следующих мест:
- `C:\Projects\dorama_alem`
- `C:\Users\alina\Documents\dorama_alem` (если Documents доступен на английском)
- `C:\dev\dorama_alem`
- `D:\Projects\dorama_alem`

### Шаг 2: Скопируйте проект

**Вариант A: Через проводник Windows**
1. Закройте все IDE и редакторы
2. Скопируйте всю папку `dorama_alem` в новое место
3. Удалите старую папку (опционально)

**Вариант B: Через командную строку**
```cmd
# Создайте новую директорию
mkdir C:\Projects

# Скопируйте проект
xcopy "C:\Users\alina\OneDrive\Документы\Dorama\dorama_alem" "C:\Projects\dorama_alem" /E /I /H /Y

# Перейдите в новую директорию
cd C:\Projects\dorama_alem
```

### Шаг 3: Обновите пути в IDE

1. Откройте проект в новой директории
2. Если используете VS Code или Android Studio, просто откройте новую папку
3. Flutter автоматически определит новое местоположение

### Шаг 4: Очистите и пересоберите

```bash
# Очистите проект
flutter clean

# Получите зависимости
flutter pub get

# Попробуйте запустить
flutter run
```

## Альтернативное решение (временное)

Если вы не можете переместить проект, можно временно отключить Material 3:

1. Откройте `lib/main.dart`
2. Измените `useMaterial3: true` на `useMaterial3: false`
3. Это отключит некоторые новые эффекты, которые требуют компиляции шейдеров

**Однако это не рекомендуется**, так как Material 3 предоставляет лучший UI/UX.

## Проверка

После перемещения проекта, попробуйте:
```bash
flutter doctor
flutter clean
flutter pub get
flutter run
```

Если все работает, значит проблема решена!
